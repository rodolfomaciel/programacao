#Terminologia Linux

##Termos linux
- kernel - o código núcleo que comanda o hardware e faz a interface entre aplicativos e hardware.
- distro - uma combinação de programas que juntamente com o kernel formam um sistema operacional. Exemplos: OpenSUSE, Ubuntu, Fedora, Gentoo, Slacware.
- boot loader - programa que inicia o sistema operacional. Exemplos: GRUB, ISOLINUX
- Services - são programas que rodam processos no background. Exemplos: httpd, nfsd, ntpd, ftpd, named.
- Sistema X Window - é um kit de ferramentas para construção de interface de usuário.
- Ambiente de trabalho - é exatamente a interface que o usuário opera. Exemplos: Gnome, KDE, XFCE, Fluxbox.
- Linha de comando - é um programa que recebe comandos para executar no sistema.
- Shell - é um intepretador de linha de comando para executar tarefas. Exemplos: bash, tcsh, zsh.
- Sistema de arquivos - são formas diferentes usadas por um sistema operacional de armazenar os arquivos nas partições de discos. Exemplos: ext4, XFS, JFS, Btrfs, NTFS, ubifs e assim por diante. Veja mais em: http://bit.ly/2cOxm3X e http://refspecs.linuxfoundation.org/fhs.shtml e http://bit.ly/2cu0aOF
- Partições - são partes do disco rigido.
